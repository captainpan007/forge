## 这个项目是什么

一个**儿子真的能玩**的电子玩具。

外观：一个塑料盒，盒上有一个 6cm 大的彩色蘑菇按钮。
行为：按一下，盒子随机播放一段声音——可能是爸爸亲口录的"宝宝最棒"，可能是动物叫，可能是儿歌片段。
内核：树莓派 4B 跑 Python，监听按钮，触发音频播放。

## 为什么从这里开始

26 个月的孩子最爱"按下去 → 有反馈"的因果游戏。这个项目同时满足：

1. **给爸爸学习**：覆盖了嵌入式入门 80% 的核心知识点（Linux / Python / GPIO / 电学 / 音频 / 自启动）；
2. **给儿子玩耍**：一周后他能拿到真正喜欢的玩具；
3. **安全可控**：完整密封后没有暴露元件，符合幼儿安全要求；
4. **门槛友好**：不需要焊接，不需要示波器，杜邦线直插即可。

## 5 个阶段

<PhaseTimeline project="soundbox" />

### Phase 1 · 心智模型（~2 小时）
不连任何线，先建立"嵌入式是什么"的脑内模型。这一阶段结束时你能看着 Pi 4 指出"这根是 GND，那根是 GPIO"。

**节点**：N01 SBC vs MCU / N02 Linux 30 秒 / N03 命令行 5 命令 / N04 电压·地·电流 / N05 GPIO 是什么 / N06 Pi 4 引脚图 / N07 面包板原理

### Phase 2 · 第一次握手（~2 小时）
让 Mac 和 Pi 通过 SSH 远程握手。从此 Pi 不需要显示器键盘。

**节点**：N08 SD 卡烧录 / N09 SSH 概念 / N10 第一次 SSH / N11 apt update / N12 nano 编辑器

### Phase 3 · 软件能力（~3 小时）
让 Python 在 Pi 上跑起来，并能控制 GPIO、监听按钮、播放音频。

**节点**：N13 Python 入门 / N14 import & pip / N15 文件路径 / N16 gpiozero 第一次 / N17 Button 类 / N18 事件驱动 / N19 pygame.mixer / N20 random

### Phase 4 · 硬件实操（~3 小时）
真把按钮接到 Pi，调通"按下打印"——这是软件 + 硬件第一次连通的"啊哈时刻"。

**节点**：N21 上拉电阻 / N22 按钮内部 / N23 杜邦线技巧 / N24 接按钮到 Pi / N25 调试打印

### Phase 5 · 给儿子用（~2 小时 + 永远）
集成 + 自启动 + 密封 + 给儿子玩。

**节点**：N26 录音 / N27 完整集成 / N28 systemd / N29 喇叭安全 / N30 密封盒 / N31 电池安全 / N32 ship it
