## What · 杜邦线 3 件事

### 公头 vs 母头

杜邦线一端是"插针"（公头 male）或"插孔"（母头 female）—— 4 种组合：

| 线 | 用途 |
|---|---|
| 公-公 (male-male) | 面包板 ↔ 面包板（两端插孔） |
| 母-母 (female-female) | Pi GPIO ↔ 模块的针脚（两端都是针） |
| **公-母 (male-female)** | **Pi GPIO ↔ 面包板** ⭐ 发声盒最常用 |
| 母-公 = 同上 | 同上 |

**怎么记：**
- Pi GPIO 针脚 = **公**（突出来的针）→ 需要**母头**接它
- 面包板的孔 = **母**（凹的）→ 需要**公头**插它
- 所以 **Pi ↔ 面包板** = **母-公 杜邦线**

### 颜色约定（业界共识）

```
红色 → 电源正极（5V / 3.3V / VCC）
黑色 → 地（GND）
黄色 → 时钟信号（SCL / CLK）
蓝色 → 数据信号（SDA / MOSI / MISO）
其他 → GPIO 信号（颜色随意但保持项目一致）
```

**不是迷信，是工程纪律：**
- 你 3 周后看自己的接线 —— 凭颜色 5 秒辨认；不守约定 = 全靠猜
- 团队协作 —— 别人 30 秒读懂你的电路
- 调试 —— 一眼看出"红线接到 GND 了" = 接错

**发声盒最低要求：** 红 = 信号 / 黑 = GND（其他颜色随便）。

### 长期稳定性 4 招

按"够用 → 完美"排序：

**招 1：插紧 + 不拉扯**（最低门槛）

接好之后，**轻拉每根线确认插紧**。Pi 针脚松的话杜邦线进半截就拔出来 = 接触不良 = 灵异故障。

**招 2：束线带 / 缠绕固定**

杜邦线很轻，碰一下就晃。**用束线带或胶带把多根线捆成一束**，整体固定到面包板或外壳，减少单根受力。

**招 3：热缩管 + 烙铁焊接**（玩具级稳定）

发声盒给小孩玩 = 必然摔 = 杜邦线必松。**真要稳定**：

1. 剪短杜邦线 → 露出铜芯
2. 用烙铁焊到按钮针脚上
3. 套热缩管 → 用热风枪/打火机加热缩紧 → 焊点不暴露

焊接 + 热缩 = 几百次摔打都不会松。

**招 4：完全用接线端子台**（工业级）

把杜邦线全换成螺丝端子或弹簧端子的接线柱（淘宝 20 元一套），用螺丝拧紧。完全不依赖物理插紧。**发声盒过度，机器人项目用。**

## Try · 动手验证你真懂了

**实际套件检查**：

1. 翻你套件里的杜邦线包装 —— 看有几种类型（通常 3 种：公公 / 母母 / 公母 各 10 根）
2. 拿一根公-母线，**两端分别试试**：
   - 母头那端插进 Pi GPIO 17 针脚 → 应该能稳稳套上
   - 公头那端插进面包板任意一孔 → 应该插进去不松
3. 拿一根**公-公**线试着插 Pi GPIO 针脚 —— **插不进去**（两针互顶）。这就是公母不匹配。

理解物理约束 → 选线前 5 秒思考"两端是什么" = 不再接错。
