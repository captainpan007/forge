## Why · 你为什么要学这一节

发声盒一共 4 个能力：
1. 检测按钮（N17 完成）
2. 事件驱动（N18 完成）
3. **播放音频**（N19 这一节）
4. 随机选音频（N20）

这一节兑现承诺：**让 Pi 发出声音**。

为什么选 pygame.mixer：
- 比 `subprocess.run(["mpg321", "a.mp3"])` 更可控（音量、暂停、淡入淡出）
- 跨平台（同代码 Mac / Pi 都能跑，方便先在 Mac 调）
- 学习成本低 —— 3 行 API
- pygame 是 Python 最稳定的多媒体库之一（20+ 年历史）

## What · 5 步播第一首歌

### 准备清单

- ✅ Pi 4 + 一个 USB 音箱（套件里那个）或 3.5mm 音箱
- ✅ 至少一个 mp3 文件（去 freesound.org 下个免版税的"喵叫"）
- ✅ N14 学过的 pip3 安装能力

### 步骤 1：装 pygame

```bash
ssh pi@raspberrypi.local

# 装到用户级
pip3 install --user pygame
# (大约 30 秒，30MB 左右)

# 验证
python3 -c "import pygame; print(pygame.version.ver)"
# 应该输出 "2.x.x"
```

### 步骤 2：插喇叭 + 选输出

**USB 喇叭：** 直接插 Pi 的 USB 口，Pi OS 默认自动选 USB 作为输出。

**3.5mm 喇叭：** 插 Pi 的 3.5mm 接口，可能需要手动切：

```bash
# 查可用输出
aplay -l

# 临时测试声音（白噪声）
speaker-test -c 2 -t wav -l 1
# 喇叭应该响一下 "Front left, Front right"
```

如果没声音，**右上角任务栏的小喇叭图标**点开切换 audio device，或：

```bash
# 强制走 3.5mm
sudo raspi-config
# → Advanced Options → Audio → Force 3.5mm
```

### 步骤 3：3 行播音频

```python
# play_test.py
import pygame

pygame.mixer.init()
pygame.mixer.music.load("/home/pi/soundbox/sounds/meow.mp3")
pygame.mixer.music.play()

# 等播完
import time
while pygame.mixer.music.get_busy():
    time.sleep(0.1)
```

**核心 3 行：** `init() → load(file) → play()`。

`get_busy()` 检测是否还在播 —— 如果不等，程序立刻退出，喇叭没响完就被切断。

### 步骤 4：把音量、循环加进来

```python
import pygame

pygame.mixer.init()

# 装载
pygame.mixer.music.load("/home/pi/soundbox/sounds/meow.mp3")

# 音量 0-1
pygame.mixer.music.set_volume(0.7)        # 70% 音量

# 播放
pygame.mixer.music.play(loops=0)          # 播 1 次（loops=-1 无限循环）

# 控制
import time
time.sleep(2)
pygame.mixer.music.pause()                # 暂停
time.sleep(1)
pygame.mixer.music.unpause()              # 继续
time.sleep(2)
pygame.mixer.music.stop()                 # 停
```

### 步骤 5：跟按钮事件结合 —— 发声盒主体诞生

```python
# soundbox_v1.py — 发声盒最小可行版本
from gpiozero import Button
from signal import pause
from pathlib import Path
import pygame
import random

# 配置
BUTTON_PIN = 17
SOUND_DIR = Path(__file__).parent / "sounds"

# 初始化
pygame.mixer.init()
pygame.mixer.music.set_volume(0.8)

# 找音频
SOUNDS = list(SOUND_DIR.glob("*.mp3"))
assert SOUNDS, f"没找到音频 — 把 mp3 放进 {SOUND_DIR}"

# 事件回调
def play_random():
    chosen = random.choice(SOUNDS)
    print(f"播放：{chosen.name}")
    pygame.mixer.music.load(str(chosen))
    pygame.mixer.music.play()

# 按钮
button = Button(BUTTON_PIN, bounce_time=0.1)
button.when_pressed = play_random

print(f"发声盒就绪 — {len(SOUNDS)} 个音频。按按钮试试。")
pause()
```

**31 行 = 一个发声盒 MVP**。

### Pi 上常见音频坑

1. **音量为 0** —— 系统音量在桌面右上角小喇叭，命令行 `alsamixer` 调
2. **USB 喇叭没自动选** —— `pavucontrol`（先 `sudo apt install pulseaudio-utils`）改默认输出
3. **mp3 播一半卡** —— 文件损坏，换一个；或者 SD 卡读取慢，换张快卡
4. **静音了** —— `amixer set Master unmute`

### 升级：让按钮按下不打断当前播放（发声盒更人性）

```python
def play_random():
    if pygame.mixer.music.get_busy():
        return  # 正在播 — 忽略这次按下
    chosen = random.choice(SOUNDS)
    pygame.mixer.music.load(str(chosen))
    pygame.mixer.music.play()
```

或者反过来 —— **每次按都从头打断重播**（儿童玩具风格）：

```python
def play_random():
    pygame.mixer.music.stop()    # 强制停
    chosen = random.choice(SOUNDS)
    pygame.mixer.music.load(str(chosen))
    pygame.mixer.music.play()
```

哪种更适合你儿子？观察他玩了一周决定。
