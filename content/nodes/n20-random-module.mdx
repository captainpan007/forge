## What · random 三件套

### `random.choice` —— 从列表挑一个

```python
import random

sounds = ["meow.mp3", "woof.mp3", "moo.mp3"]
chosen = random.choice(sounds)
print(chosen)            # 三个里随机一个
```

**最常用。** 发声盒的灵魂这一行。

### `random.randint(a, b)` —— 随机整数

```python
n = random.randint(1, 6)     # 1 到 6 之间（含两端）
print(n)                     # 1 / 2 / 3 / 4 / 5 / 6 之一
```

**注意：** randint **两端都含**。`randint(1, 6)` 包括 1 和 6。

### `random.shuffle(列表)` —— 原地打乱

```python
sounds = ["a", "b", "c", "d"]
random.shuffle(sounds)        # 原地修改
print(sounds)                 # 可能 ["c", "a", "d", "b"]
```

**坑：** `shuffle` 不返回新列表，**直接修改原列表**。要保留原序用 `random.sample(list, len(list))` 拿乱序副本。

### 发声盒进阶：避免连续重复

```python
# 简单版：完全随机（可能连续 2 次同一首）
def play_random():
    chosen = random.choice(SOUNDS)
    play(chosen)

# 进阶版：避免上次播过的
last_sound = None
def play_random_no_repeat():
    global last_sound
    available = [s for s in SOUNDS if s != last_sound]
    chosen = random.choice(available) if available else random.choice(SOUNDS)
    last_sound = chosen
    play(chosen)

# 最佳版：洗牌一轮，轮播完再洗
queue = []
def play_random_shuffled():
    global queue
    if not queue:
        queue = SOUNDS[:]       # 拷贝
        random.shuffle(queue)
    chosen = queue.pop()
    play(chosen)
```

**给儿童玩具推荐"洗牌一轮"模式** —— 每首音频都会出现，不会连续重复，可预期但不死板。

### 伪随机 vs 真随机（一分钟科普）

`random` 模块用的是「伪随机数生成器」（Mersenne Twister）。**伪 = 看起来随机但本质是数学算法**：

```python
random.seed(42)
print(random.random())     # 0.6394...
print(random.random())     # 0.0250...

random.seed(42)            # 同样种子
print(random.random())     # 0.6394... ← 完全一样！
print(random.random())     # 0.0250... ← 完全一样！
```

**这就是为什么密码学 / 抽奖系统不能用 `random`** —— 知道种子能预测全部输出。它们用 `secrets` 模块（基于硬件熵 + 加密算法）。

但**发声盒、游戏、模拟器**用 `random` 完全够 —— 你不在乎能不能被预测，只要看起来够"乱"。
