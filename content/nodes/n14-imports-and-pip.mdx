<HandsOnHook scene="第一次需要别人写的代码">

你想让 Pi 控制 LED，搜到一段代码：

```python
from gpiozero import LED
led = LED(17)
led.on()
```

兴致勃勃 SSH 进 Pi 敲完，按回车 —— **ModuleNotFoundError: No module named 'gpiozero'**。

10 分钟读完你会知道：Python 自带的"标准库"只够基础操作，**90% 的项目都要装第三方库**。pip 就是装库的工具，跟 N11 的 apt 是亲兄弟。

</HandsOnHook>

## Why · 你为什么要学这一节

发声盒所有真正"动硬件"的能力都来自第三方库：

- `gpiozero` —— 控制 GPIO 点亮 LED / 读按钮（N16, N17）
- `pygame` —— 播放音频（N19）
- `random` —— 随机选音频（N20，标准库自带）

不会 pip = 你看到再好的教程都装不上。10 分钟搞定 pip + 顺手懂 venv（虚拟环境）。

<Callout type="tip" icon="🎯">
  **学完这一节你能做到：** 看到任何 Python 教程贴 `pip install xxx`，你能在 Pi 上装好并 import 进自己的代码。
</Callout>

## What · import 和 pip 是两件事

### import：在代码里"借用"别人的代码

```python
import random           # 整个 random 模块都借过来用 random.choice()
from gpiozero import LED  # 只借 gpiozero 里的 LED 类
import time as t        # 起别名（少见，但能用 t.sleep()）
```

模式：**`import 库名` 或 `from 库名 import 类/函数`**。

但**库不会凭空存在** —— 你得先装。这就是 pip 的工作。

### pip：Python 的"App Store"

```bash
pip3 install gpiozero          # 装 gpiozero
pip3 install pygame            # 装 pygame
pip3 install requests==2.31.0  # 装指定版本
pip3 list                      # 看已装啥
pip3 show gpiozero             # 看某个库的详情
pip3 uninstall gpiozero        # 删
```

**注意：** Pi 上 `pip` 可能指 Python 2（已废弃），**永远用 `pip3`** 确保是 Python 3 的 pip。

### Pi 新手第一坑：`sudo pip3 install`

```bash
sudo pip3 install gpiozero    # ⚠ 不推荐！
```

为什么不推荐：

- **污染系统 Python**：装到 /usr/lib/python3/ 里，可能跟 Pi OS 自带的库冲突
- **后续删除困难**：装错了不知道清哪
- **安全风险**：第三方库代码以 root 运行

**推荐做法 1：用户级安装（--user）：**

```bash
pip3 install --user gpiozero  # ✓ 装到 ~/.local/，干净
```

**推荐做法 2：用虚拟环境（venv）：**（更专业 / 多项目时必备）

```bash
# 在你的项目目录下：
cd ~/soundbox
python3 -m venv venv          # 创建 venv 文件夹
source venv/bin/activate      # 激活（命令行前面会显示 (venv)）
pip3 install gpiozero pygame  # 装在 venv 里，跟其他项目隔离
# ... 写代码 / 跑脚本 ...
deactivate                    # 退出虚拟环境
```

<Callout type="info" icon="🧠">
  **Pi 上 gpiozero 通常预装。** Raspberry Pi OS 出厂带 RPi.GPIO + gpiozero。如果 import 失败可能是 Pi OS 版本太老 — 但 N11 的 apt update + upgrade 会把它升到新版。
</Callout>

## Try · 动手验证你真懂了

**SSH 进 Pi 真做一次**：

```bash
# 1. 看 gpiozero 已装没（Pi OS 预装）
python3 -c "import gpiozero; print(gpiozero.__version__)"
# 应该输出版本号像 "2.0.1"

# 2. 装一个新库试试（pip3 而不是 pip）
pip3 install --user pyfiglet   # 装一个 ASCII art 库

# 3. 用一下
python3 -c "import pyfiglet; print(pyfiglet.figlet_format('Hello'))"
# 应该看到 ASCII 字符画

# 4. 看你装了哪些用户级库
pip3 list --user

# 5. 删掉这个测试库
pip3 uninstall pyfiglet -y
```

跑通 = 你掌握了 Python 装库全流程。
