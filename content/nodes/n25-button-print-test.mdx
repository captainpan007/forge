## What · 5 行测试代码

### 极简版

```python
# button_test.py
from gpiozero import Button
from signal import pause

button = Button(17, bounce_time=0.1)
button.when_pressed = lambda: print("按了！")
print("等你按按钮...")
pause()
```

**整个文件 6 行**。SSH 进 Pi 跑：

```bash
ssh pi@raspberrypi.local
cd ~/soundbox          # 或者你的项目目录
nano button_test.py    # 粘贴上面的代码
# Ctrl+O 保存，Ctrl+X 退出
python3 button_test.py
```

终端显示 "等你按按钮..."。**按下大蘑菇按钮** → 应该看到 "按了！"。多按几次：

```
等你按按钮...
按了！
按了！
按了！
^C                      ← Ctrl+C 退出
```

**🎉 这就是发声盒 v0 — 物理按钮 → Python 反应**。

### 升级版：带计数 + 时间戳

```python
# button_test_v2.py
from gpiozero import Button
from signal import pause
from datetime import datetime

button = Button(17, bounce_time=0.1)
count = 0

def on_press():
    global count
    count += 1
    timestamp = datetime.now().strftime("%H:%M:%S")
    print(f"[{timestamp}] 第 {count} 次按下")

button.when_pressed = on_press
print("按钮测试 — Ctrl+C 退出")
pause()
```

跑起来：

```
按钮测试 — Ctrl+C 退出
[14:23:05] 第 1 次按下
[14:23:07] 第 2 次按下
[14:23:08] 第 3 次按下
```

**有时间戳 + 序号** = 调试体验从"不知道是不是 bounce 误触"升级到"看得见每次按的时间间隔"。

## 调试三步法（嵌入式工程师的核心技能）

代码跑了，**按按钮没反应**。是按钮坏了、Pi 坏了、代码 bug、还是接线松了？随机猜会让你浪费几小时。**用三步法 30 秒定位：**

### 第 1 步：硬件层 — 万用表通断

**核心问题：** 物理上电流真的从 GPIO 17 经过按钮流到 GND 吗？

```
Pi 断电
万用表通断档：表笔接 Pi 物理 11（GPIO 17）+ Pi 物理 6（GND）
按按钮：响 = 硬件通；不响 = 接线问题（停在这里修）
```

<Callout type="tip" icon="🔧">
  万用表是嵌入式工程师的"听诊器"。**淘宝 50 块买一把数字万用表** —— 通断档 + 直流电压档 + 电阻档够你用 10 年。
</Callout>

### 第 2 步：底层 API — gpiozero 在读什么

**核心问题：** 硬件通了，但 Python 真的看到 GPIO 17 的变化吗？

```python
from gpiozero import DigitalInputDevice
import time

g = DigitalInputDevice(17, pull_up=True)
for _ in range(20):
    print(g.value)            # 应该看到 1 → 1 → 1 → 0 (按下时) → 1
    time.sleep(0.5)
```

看到值随按下变化 → GPIO 读取正常，Button 类外是 OK 的。

### 第 3 步：业务代码 — Button 类 + 回调

**核心问题：** 上面都对，那就是 Button 类初始化 / 回调注册的问题。

最常见错误：

```python
# 错 1：忘了 pause() — 进程立刻退出
button.when_pressed = on_press
# 没 pause()，按钮按了也没回调

# 错 2：回调里出错被静默吞掉
def on_press():
    play_sound(undefined_var)   # ← 抛异常 gpiozero 默认吃掉

# 错 3：bounce_time 没设 — 一次按触发多次
button = Button(17)             # 默认 bounce_time = None
```

**调试技巧**：回调里加 try/except 显式打印：

```python
def on_press():
    try:
        # 你的逻辑
        play_random()
    except Exception as e:
        print(f"回调出错: {e}")
```
