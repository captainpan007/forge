<HandsOnHook scene="从桌面到玩具">

发声盒所有功能就绪 —— 但**桌上还是一堆杜邦线 + 面包板 + Pi**。让儿子玩？他会立刻拽线。

要变成"真玩具"必须把所有东西装进**一个塑料盒子**：

- 大蘑菇按钮露在盒子表面（儿子能按）
- USB 喇叭固定在盒内（声音从孔出来）
- Pi 藏在盒内（孩子摸不到，不会乱拔）
- 电源线 / 开关在背面（你方便维护）

60 分钟教你**所有打孔位置 + 走线技巧 + 防摔加固**。这一节做完发声盒就**像个玩具**而不是实验。

</HandsOnHook>

## Why · 你为什么要学这一节

工程师做出能跑的项目只是 50%，剩下 50% 是"它能不能交付到真用户手里"。**外壳是玩具/产品/实验的分界线**。

发声盒装盒后：
- 儿子能扔进玩具箱混着 lego 一起玩
- 你不在旁边他自己能玩 30 分钟
- 摔几次不散架
- 内部线藏起来 —— 安全 + 美观

<Callout type="tip" icon="🎯">
  **学完这一节你能做到：** 拿到任何项目 + 一个塑料盒，规划好打孔位置 + 走线，60 分钟内装出能用的成品。
</Callout>

## What · 6 步装盒

### 准备：盒子选型

**推荐：约 15cm × 10cm × 7cm 透明塑料密封盒**（淘宝"防水接线盒"，¥15）。

**为什么这个尺寸：**
- 装得下 Pi 4 + USB 喇叭 + 杜邦线 + 18650 电池（N31 加）
- 不至于太大让 2 岁小孩抱不动
- 透明 = 儿子能"看见里面"满足好奇

**关键：** 选**带螺丝的盒子**（不是卡扣式）—— 卡扣久了会松，螺丝可以反复拆装维护。

### 步骤 1：测量 + 标记位置

铺平盒盖（顶面）—— 在里面摆好按钮、喇叭模拟位置，**铅笔在外侧标圆心**：

```
┌──────────────────────────┐
│                          │
│      ●                   │   ← 大蘑菇按钮圆心
│   (中心偏左, 6cm 直径)    │
│                          │
│              ▓ ▓ ▓       │   ← 喇叭出声口（多个小孔阵列）
│              ▓ ▓ ▓       │
│                          │
└──────────────────────────┘
   ↑ 盒盖顶视图
```

**为什么不打一个大孔放喇叭：** 强度差，多个小孔（阵列 5×5）声音照样透 + 盒盖牢固。

### 步骤 2：打按钮大孔（直径 ~22mm）

大蘑菇按钮的螺纹通常 22mm。

**工具：步进钻头**（淘宝 ¥30 一套 4-32mm）：

1. 在标记的圆心用尖锥**戳个小坑**（防止钻头打滑）
2. 步进钻头从小往大钻 —— 一档一档扩
3. 钻到 22mm 停下
4. 用细砂纸打磨孔边毛刺

**没有步进钻：** 用 3mm 钻头钻一圈小孔（沿圆周），然后用钳子掰掉中间，砂纸打磨。粗糙但能用。

### 步骤 3：打喇叭出声孔阵列

3mm 钻头钻一片 5×5 网格（间距 5mm）：

```
○ ○ ○ ○ ○
○ ○ ○ ○ ○
○ ○ ○ ○ ○
○ ○ ○ ○ ○
○ ○ ○ ○ ○
```

声波 80% 通过这片孔阵 — **跟一个大孔效果相当 + 盒盖强度不损失**。

### 步骤 4：打侧面孔（电源 + 开关）

侧面打 2 个孔：

- **USB-C 电源孔**（10mm 椭圆）—— Pi 4 USB-C 接口对外
- **小开关孔**（10mm 圆）—— 给 Pi 电源加个物理总开关（N31 加电池后必备）

### 步骤 5：装按钮 + 喇叭

**按钮安装：**

1. 按钮从盒盖**外侧**穿过去（蘑菇头朝外）
2. 内侧拧上塑料螺母固定
3. 用力按几下确认不晃

**喇叭安装：**

1. 把 USB 喇叭主体用**双面胶 + 热熔胶**固定在盒盖内侧
2. 喇叭出声口对准你打的孔阵
3. 试听 — 没明显闷音就行

### 步骤 6：装 Pi + 走线 + 收尾

```
┌──────────────────────────┐
│      ▓ ▓ ▓ ▓ ▓           │  ← 喇叭出声孔
│      ▓ ▓ ▓ ▓ ▓           │
│      ▓ ▓ ▓ ▓ ▓           │
│                          │
│  ▶ Pi 4 (用螺丝/胶贴底)   │
│      │                   │
│      │ 杜邦线（束线带捆）  │
│      │                   │
│  ▶ USB 喇叭模块           │
│                          │
└──────────────────────────┘
   ← 侧面 USB-C 电源孔
```

**关键：**
- Pi 4 用 M2.5 螺丝固定到盒底（防摔位移）—— 淘宝 ¥5 一套
- 杜邦线用**束线带每 3cm 捆一次** —— 减少抖动松动
- 盒盖盖回去前**测一次按钮**确认能正常按下（盒盖压住按钮就废了）

## Try · 动手验证你真懂了

**真装一次盒**（这一节是动手节，必须实操）：

1. ✅ 准备：塑料盒 + 步进钻头 + 双面胶 + 热熔胶 + M2.5 螺丝
2. ✅ 测量并标记位置（按钮 + 喇叭 + 电源孔）
3. ✅ 钻按钮大孔（22mm）
4. ✅ 钻喇叭出声孔阵（5×5）
5. ✅ 钻侧面 USB-C 孔
6. ✅ 装按钮 + 喇叭 + Pi 固定
7. ✅ 杜邦线用束线带捆扎
8. ✅ 盖盒盖测试按钮 — 正常响

完成 = 你的发声盒**像一个真玩具了**。给儿子玩 10 分钟看他反应。
