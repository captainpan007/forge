## Why · 你为什么要学这一节

**面包板**（breadboard）—— 你套件里那块白色的板子。**它的内部布线 90% 新手不知道，所以 90% 第一次接电路的人接错。**

不是你不够聪明 —— 是从外观看不出"哪两个孔是连通的"。10 分钟看完这一节，下次插线你就有了"X 光视觉" —— 透过塑料壳看到内部金属条的连法。发声盒的按钮 + 喇叭电路全靠它。

### 内部布线 —— 看不见但极重要

```
   ─── + ─── + ───  ← 顶部红色长线 (Power Rail)：所有孔横向相通
   ─── - ─── - ───  ← 顶部蓝色长线 (Ground Rail)：所有孔横向相通

   ┃ ┃ ┃ ┃ ┃ ┃ ┃    ← 上半部分：5 个孔为一组，纵向相通
   ┃ ┃ ┃ ┃ ┃ ┃ ┃       (a-b-c-d-e 一组，整条不通)
   ━━━━━━━━━━━━━━━━━━ ← 中间隔离沟（不导电，把上下分开）
   ┃ ┃ ┃ ┃ ┃ ┃ ┃    ← 下半部分：5 个孔为一组，纵向相通
   ┃ ┃ ┃ ┃ ┃ ┃ ┃       (f-g-h-i-j 一组)

   ─── + ─── + ───  ← 底部红色长线
   ─── - ─── - ───  ← 底部蓝色长线
```

**关键规则：**
1. **红线长条（Power Rail）：** 顶/底两条横向长线，整条相通。一般接电源 +。
2. **蓝线长条（Ground Rail）：** 顶/底两条横向长线，整条相通。一般接 GND。
3. **中间主区：** 5 个孔为一组**纵向**相通。比如 a1-b1-c1-d1-e1 是一组，但 a1 和 a2 不通。
4. **中间凹沟：** 上下两半隔离 —— a 那一组 (a-b-c-d-e) 和 f 那一组 (f-g-h-i-j) 不直接连。

<Callout type="info" icon="🧠">
  **为什么有那道沟？** 给"双列直插芯片"（DIP）用 —— 芯片左右两排针脚分别落在沟两侧的不同孔组里，左右针脚自然不会短路。
</Callout>

### 实战：怎么接一个 LED + 限流电阻

**目标：** Pi 的 GPIO 17 → 限流电阻 220Ω → LED → GND

```
Pi GPIO 17 (BCM 17 / 物理 11) ──── 红色杜邦线 ──┐
                                             │
                              面包板 a1 孔 ───┘
                              ↓
                              c1 孔（同组，相通）
                              ↓
                              电阻一端插 c1，另一端插 c5
                              ↓
                              c5 跟 a5、b5、d5、e5 同组
                              ↓
                              LED 长腿(+) 插 b5，短腿(-) 插 b6
                              ↓
                              b6 跟 b7-b10 同组
                              ↓
                              黑色杜邦线从 a10 ──── Pi 任意 GND 针脚
```

**这套接完，gpiozero `LED(17).blink()` 跑起来 LED 就闪了。** N16 会带你完整跑一次。

### 安全 3 条铁律

1. **跨沟接线没意义 ——** 上下两半被沟隔开，跨过去的话两侧都得各接一根线
2. **同组孔互通 ——** 一个 5 孔组里你已经接了一根线，再插同组任意孔还是同一个电节点
3. **红蓝长条整条都通 ——** 不要在红线长条上同时插两个不同电压源（直接短路）
