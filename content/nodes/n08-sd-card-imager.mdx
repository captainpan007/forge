<HandsOnHook scene="开箱后第一难关">

Pi 4 拿到了。打开包装一看 —— **里面没硬盘**。你心里咯噔："那系统装哪儿？"

翻包装，发现一张指甲盖大的塑料卡片，写着 "microSD 32GB"。教程说"把这张卡变成 Pi 的'硬盘'"。

20 分钟后，你的 Pi 第一次能开机了 —— **关键在于一个免费工具，叫 Raspberry Pi Imager**。这一节教你 5 步搞定。

</HandsOnHook>

## Why · 你为什么要学这一节

Pi 4 **没有内置硬盘**。它的"硬盘"就是你塞进去的那张 microSD 卡。这张卡上必须有一套完整的操作系统（Raspberry Pi OS），Pi 通电才能启动。

把系统"装"进 SD 卡的过程叫**烧录**（imaging / flashing）。Pi 官方做了一个超友好的工具 —— **Raspberry Pi Imager** —— 一键搞定。

<Callout type="tip" icon="🎯">
  **学完这一节你能做到：** 拿到任何一张 microSD 卡，20 分钟变成 Pi 可启动的系统盘，里面预设好你的 WiFi 密码和 SSH 开关。
</Callout>

## What · 5 步完成烧录

### 步骤 1：下载 Raspberry Pi Imager

去官网下载（Mac / Win / Linux 都有）：

```
https://www.raspberrypi.com/software/
```

Mac 双击 `.dmg` 拖到 Applications 即可。100MB 左右，国内代理快的话 1 分钟下完。

### 步骤 2：插 microSD 卡

把 32GB（或更大）的 microSD 卡用读卡器插到你 Mac 上。如果 Mac 没 SD 卡槽，需要一个 USB-A / USB-C 读卡器（套件里通常带一个，淘宝 20 元）。

⚠ **卡里数据会全部抹掉。** 提前确认这张卡里没有重要文件。

### 步骤 3：打开 Imager 选三件事

Imager 主界面有 3 个大按钮：

```
┌─────────────────┬─────────────────┬─────────────────┐
│  CHOOSE DEVICE  │  CHOOSE OS      │  CHOOSE STORAGE │
└─────────────────┴─────────────────┴─────────────────┘
```

**按顺序点：**

1. **CHOOSE DEVICE** → 选 **Raspberry Pi 4**
2. **CHOOSE OS** → 选 **Raspberry Pi OS (64-bit)**（推荐 Pi 4，性能更好）
3. **CHOOSE STORAGE** → 选你插进去的那张 microSD（认错盘可能毁掉硬盘上的数据，**一定看清楚是 32GB 那张**）

<PredictThenReveal
  question="如果 CHOOSE STORAGE 里我手抖选了我 Mac 的内置 1TB SSD，点 WRITE 会发生什么？"
  hint="Imager 是把镜像直接写进选中的盘，不管那盘原来是什么..."
  answer="毁掉你 Mac。Imager 不识别盘是不是系统盘，它会把 Raspberry Pi OS 镜像直接写进去 —— 你 Mac 上的所有数据会被覆盖、系统无法启动。所以选 STORAGE 时一定看容量（microSD 32GB ≠ 1TB SSD），看清楚再点。"
/>

### 步骤 4：⚙ 高级设置（这一步决定你不用接显示器）

点 WRITE 之前，**先点右下角齿轮 ⚙**（或按 `⌘ + Shift + X`）打开高级设置：

```
☑ Set hostname:        raspberrypi.local
☑ Enable SSH:          [Use password authentication]
   ─ Username:         pi
   ─ Password:         (你自己设一个，记住)
☑ Configure wireless LAN:
   ─ SSID:             (你家 WiFi 名)
   ─ Password:         (你家 WiFi 密码)
   ─ Wireless country: CN
☑ Set locale settings:
   ─ Timezone:         Asia/Shanghai
   ─ Keyboard layout:  us
```

<Callout type="warning" icon="🔥">
  **这一步是关键中的关键。** 如果跳过，烧好的 Pi 第一次开机时找不到 WiFi、SSH 也关着 —— 你必须临时找显示器键盘配置。**勾上 SSH + 填好 WiFi，Pi 开机自动联网 + 自动开 SSH，你直接 Mac 远程进去。**
</Callout>

### 步骤 5：WRITE → 等 10-15 分钟

回主界面点 **WRITE**。Imager 会：

1. 从官网下载 Raspberry Pi OS 镜像（约 1GB，第一次烧需要联网）
2. 把镜像写入 SD 卡（5-10 分钟）
3. 验证写入完整（再 5 分钟）

显示 ✅ "Write Successful" 后，卡能直接拔（Imager 已经卸载它了）。

## Try · 动手验证你真懂了

**真做一次**（这一节就是动手节）：

1. 下载 Pi Imager → 装好
2. 找一张 32GB microSD + 读卡器
3. Imager → CHOOSE DEVICE: Pi 4 → CHOOSE OS: Raspberry Pi OS 64-bit → CHOOSE STORAGE: 你的 SD
4. ⚙ 高级设置：勾 SSH（用密码登录）+ 填 WiFi（你家 SSID + 密码）+ Pi 用户名/密码
5. WRITE → 等完成
6. 拔卡 → 插 Pi → 通电

**Pi 红灯常亮 + 绿灯偶尔闪烁就是正常启动状态。**
