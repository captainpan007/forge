### 完成后的物理样子

```
你的桌面应该像这样：

  [大蘑菇按钮]──黑线──[Pi 物理 6 GND]
       │
       └────橙线────[Pi 物理 11 GPIO 17]

  Pi 4 通电（红绿灯）
  按钮在桌上等你按
```

### 常见接线错误（5 大坑）

| 错误 | 症状 | 修复 |
|---|---|---|
| **物理 17 当成 GPIO 17** | 按钮没反应（你接到 3.3V 上了）| 改成物理 **11** |
| **接了 COM + NC** | 没按时一直触发，按了反而停 | 换成 COM + NO |
| **公母不匹配** | 杜邦线插不进 / 接触不良 | 母-母线接 Pi+按钮（两端都是针） |
| **GND 漏接** | 按钮无反应（电路不通） | 黑线连按钮 COM 跟 Pi 任意 GND（6/9/14/20/25/30/34/39）|
| **杜邦线松** | 时灵时不灵 | 轻按确认插紧，长期用焊接 |

## Try · 动手验证你真懂了

按照上面 6 步真做一次。**关键验证 = 第 5 步万用表通断测试**：

1. ✅ Pi 断电
2. ✅ 按钮 COM 接 Pi GND（黑线）
3. ✅ 按钮 NO 接 Pi GPIO 17（橙线）
4. ✅ 万用表通断档接 GPIO 17 + GND
5. ✅ 按按钮 → 万用表蜂鸣
6. ✅ 松开 → 万用表静音

6 步都对 = 硬件 100% 就绪。下一节 N25 跑代码看效果。
