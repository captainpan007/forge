## What · 完整 main.py（带详细注释）

### 完整代码（30 行）

```python
#!/usr/bin/env python3
"""
soundbox — 给儿子的 dad-made 发声盒
按一下按钮，随机播放一段音频
"""
from gpiozero import Button
from signal import pause
from pathlib import Path
from datetime import datetime
import pygame
import random
import sys

# ─── 配置 ─────────────────────────────
BUTTON_PIN = 17                                  # BCM 17 = 物理 11
SOUND_DIR = Path(__file__).parent / "sounds"     # 脚本同级 sounds/
VOLUME = 0.7                                     # 0.0-1.0，给小孩留余量
BOUNCE_TIME = 0.1                                # 100ms 去抖

# ─── 初始化 ───────────────────────────
pygame.mixer.init()
pygame.mixer.music.set_volume(VOLUME)

SOUNDS = sorted(SOUND_DIR.glob("*.mp3"))
if not SOUNDS:
    print(f"ERROR: 没找到 mp3 文件在 {SOUND_DIR}")
    sys.exit(1)

# "shuffle + 队列"模式 — 一轮内不重复（N20 学的）
queue = []

# ─── 事件回调 ─────────────────────────
def play_random():
    global queue
    # 队列空了 → 洗一轮
    if not queue:
        queue = SOUNDS[:]
        random.shuffle(queue)

    chosen = queue.pop()
    timestamp = datetime.now().strftime("%H:%M:%S")
    print(f"[{timestamp}] ▶ {chosen.name}  ({len(queue)} 剩余)")

    pygame.mixer.music.load(str(chosen))
    pygame.mixer.music.play()

# ─── 启动 ─────────────────────────────
button = Button(BUTTON_PIN, bounce_time=BOUNCE_TIME)
button.when_pressed = play_random

print(f"🎵 发声盒就绪 — {len(SOUNDS)} 段音频")
print(f"📍 GPIO {BUTTON_PIN} (BCM)，音量 {int(VOLUME*100)}%")
print(f"按 Ctrl+C 退出")
pause()
```

### 逐段拆解

**配置段** —— 所有可调参数集中在顶部：

```python
BUTTON_PIN = 17       # 改 GPIO 直接改这里
VOLUME = 0.7          # 调音量改这里
BOUNCE_TIME = 0.1     # 防误触改这里
```

**初始化段** —— 资源准备：

```python
pygame.mixer.init()              # 启动音频系统
SOUNDS = sorted(SOUND_DIR.glob("*.mp3"))   # 拉取所有 mp3
```

`sorted` 让顺序稳定（每次跑文件顺序一致，方便排错）。

**洗牌队列** —— 不连续重复：

```python
queue = []     # 模块级变量保存洗牌后的序列
```

第一次按按钮 → queue 空 → shuffle 一轮 → pop 第一个；后续按按钮直接 pop。一轮播完队列空了 → 自动重新洗。

**事件回调** —— 核心逻辑：

```python
button.when_pressed = play_random   # 按下时调
pause()                              # 让进程活着
```

### 错误处理（生产代码必备）

上面代码还有进步空间。**生产版加 try/except**：

```python
def play_random():
    global queue
    try:
        if not queue:
            queue = SOUNDS[:]
            random.shuffle(queue)
        chosen = queue.pop()
        print(f"▶ {chosen.name}")
        pygame.mixer.music.load(str(chosen))
        pygame.mixer.music.play()
    except Exception as e:
        # gpiozero 回调里抛异常会被静默吞掉 — 显式打印才能调试
        print(f"⚠ 错误: {e}")
```

文件损坏 / 音频系统挂了 / 队列异常 —— try/except 让你看到错误而不是"按按钮没反应"。
