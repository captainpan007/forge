## What · 安全锂电池方案

### 三层保护原则

```
锂电池 (18650 单节 3.7V)
   ↓
保护板（过充 / 过放 / 短路保护）
   ↓
TP4056 充电模块（充电管理 + USB-C 输入）
   ↓
DC-DC 升压（3.7V → 5V）
   ↓
Pi 4 USB-C
```

**别简化任何一层**。每一层都防一种事故。

### DIY 方案（不推荐但解释清楚）

如果你坚持自己接（成本 ¥40，但风险高）：

```
零件清单：
- 18650 电池一节（3.7V 3400mAh）
- 18650 电池座（带焊脚）
- 1S 锂电池保护板（淘宝 ¥5）
- TP4056 充电模块（带 Type-C 接口，¥3）
- MT3608 升压模块（3.7V → 5V/2A，¥5）
- 杜邦线 / 焊接工具
```

**接线顺序（必须按这个）：**

```
[USB-C 充电口] ──→ [TP4056] ──→ [保护板] ──→ [18650+]
                                              ↓
                  [Pi 4 USB-C] ←── [MT3608 升压] ←── [18650-]
```

⚠ **任何一根线接错 = 起火 / 爆炸**。看视频教程 + 万用表确认每根线对了再通电。

### 关键安全准则

**1. 选认证电池**

✓ 三星 / 松下 / LG 原厂 18650（淘宝 ¥30-50 一节）
✗ 山寨/不知名品牌 18650（¥5 一节）— **起火率高 100 倍**

**怎么辨认：**
- 看标签印刷质量（山寨字体歪扭）
- 看重量（真 18650 ≥ 45g，假货 30g）
- 看店铺评价 + 售后

**2. 永远用保护板**

1S 保护板（¥5）的作用：

- 过放保护：电压 < 2.5V 自动断开
- 过充保护：电压 > 4.2V 自动断开
- 短路保护：电流 > 5A 自动断开

**没保护板 = 玩具变定时炸弹**。

**3. 物理结构防压**

儿童会摔玩具。18650 被压扁 → 内部短路 → 起火。

发声盒装盒时：

- 电池放盒子**底部**（重心低 + 不容易被压到按钮区）
- 用**电池座**（不要焊在电池上）— 撞击时电池能弹出而不是变形
- 电池周围**留 5mm 缓冲间隙**（不是塞紧）

**4. 充电时不让孩子玩**

锂电池充电时温度最高 → 风险最高。

养成习惯：

- 发声盒充电时 = 大人监管 + 平整桌面 + 远离孩子
- **不能孩子拿着发声盒同时充电** = 这是儿童玩具锂电池最大风险场景

### 充电指示

Pisugar 3 / TP4056 都带 LED 灯：

```
红灯 = 充电中
蓝/绿灯 = 充满
不亮 = 没接电 或 已充满
```

**满了立刻拔 USB-C** —— 持续涓流充电会缩短电池寿命（虽然有保护板）。
