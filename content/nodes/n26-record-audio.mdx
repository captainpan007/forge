## Why · 你为什么要学这一节

发声盒一周内你儿子玩腻了同一组音频。**长期可玩性 = 你能持续供给新内容**。这一节是发声盒"内容生产线"的基础工程。

收集渠道 3 种：
- **录爸爸妈妈的声音**（最珍贵，孩子记得一辈子）
- **录家里宠物 / 家电 / 环境音**（孩子认得的真实世界声音）
- **网上找免版税音效**（动物叫声、儿歌片段）

<Callout type="tip" icon="🎯">
  **学完这一节你能做到：** 30 分钟内攒出 20+ 段高质量音频文件，命名规范，传到 Pi 待发声盒调用。
</Callout>

## What · 3 个渠道 + 1 套规范

### 渠道 1：Mac Voice Memos 录爸爸妈妈

最简单 + 最珍贵的内容：

1. 打开 Mac 自带的 **Voice Memos**（语音备忘录）
2. 按红色按钮录音（建议每段 2-5 秒：一句话、一个动物叫声、一段旋律）
3. 录完点 ✓ 完成
4. 右键 → "Export As..." → 保存为 mp3 或 m4a

**录什么内容（建议清单）：**
- "宝宝你好" / "爸爸爱你" / "妈妈爱你"
- 模仿动物：喵～ / 汪汪 / 哞～ / 嘎嘎
- 模仿交通工具：嘀嘀 / 呜呜（火车）/ 哐哐（地铁）
- 简单数数 "一、二、三"
- 一段简短儿歌（"小星星"前两句）

<Callout type="info" icon="🎙">
  **录音技巧**：嘴离麦克风 10-20cm 太近会爆音，太远声音糊。Mac 内置麦在键盘上方 — 对着屏幕中部说话最干净。
</Callout>

### 渠道 2：freesound.org 找免版税音效

**[freesound.org](https://freesound.org)** 是音效工程师的圣地：

1. 注册（免费）
2. 搜你要的（"cat meow"、"car horn"、"baby laugh"）
3. 选 **CC0 (Creative Commons Zero)** 标记的 — 完全免版税
4. 下载 wav 或 mp3

**注意：** 找 **3 秒以内的短音效**，太长不适合按钮触发。

### 渠道 3：用 Pi 自己录（进阶）

接 USB 麦克风到 Pi，用 `arecord`：

```bash
ssh pi@raspberrypi.local

# 装工具
sudo apt install alsa-utils -y

# 看可用录音设备
arecord -l

# 录 5 秒
arecord -d 5 -f cd ~/soundbox/sounds/test.wav

# 播一下听听
aplay ~/soundbox/sounds/test.wav
```

**适合：** 想让发声盒"记录儿子自己的声音"作为彩蛋。`v0.5` 进阶玩法。

### 命名约定（极其重要）

```
sounds/
├── voice-dad-hello.mp3        ← 类型-录制者-内容
├── voice-mom-loveyou.mp3
├── animal-cat-meow.mp3
├── animal-dog-bark.mp3
├── vehicle-car-horn.mp3
├── music-littlestar.mp3
└── numbers-onetwothree.mp3
```

**为什么这个命名：**
- **类型前缀** = 你想"只播动物"时一句 `glob("animal-*.mp3")` 就行
- **录制者** = 半年后你不记得是爸爸还是妈妈录的
- **小写 + 连字符** = Linux 文件系统友好（避免空格 / 中文路径问题）

### 音量归一化（30 秒搞定）

你录的爸爸妈妈声音音量不一，加上 freesound 下载的，喇叭里听起来忽大忽小 —— 难听。

**Mac 上用 ffmpeg 一键归一化**：

```bash
brew install ffmpeg          # Mac 上装 ffmpeg

cd ~/Desktop/sounds-raw      # 你的原始音频文件夹

# 批量归一化到 -16 LUFS（行业标准的"舒适音量"）
mkdir -p ../sounds-normalized
for f in *.mp3; do
    ffmpeg -i "$f" -af "loudnorm=I=-16:TP=-1.5:LRA=11" "../sounds-normalized/$f"
done
```

跑完看 `sounds-normalized/` 文件夹 —— 所有 mp3 音量一致，发声盒里听起来舒服。

### 传到 Pi（scp）

Mac 终端跑：

```bash
# 把整个文件夹拷到 Pi
scp -r ~/Desktop/sounds-normalized pi@raspberrypi.local:~/soundbox/sounds

# 拷单个文件
scp ~/Desktop/dad-hello.mp3 pi@raspberrypi.local:~/soundbox/sounds/

# 输 Pi 密码即可
```

或者用 **SFTP**（图形界面）：用 Cyberduck（Mac 免费）连 sftp://pi@raspberrypi.local，拖文件夹进去。
