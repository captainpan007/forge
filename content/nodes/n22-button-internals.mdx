## What · 按钮内部就是一对触点

### 简单按钮 vs 复杂按钮

**最简单的按钮**（面包板 tactile 按钮）：4 个针脚，**两两短接**（即 1-2 是一组，3-4 是一组）。按下时两组之间导通。

**大蘑菇按钮**（带 LED 的 6cm 大按钮）：4 个针脚，分两类：
- **2 个 = 按钮开关触点**（COM + NO 或者 COM + NC）
- **2 个 = LED 灯触点**（+ 和 -，给按钮自身的指示灯供电）

发声盒**只用开关那 2 个**。LED 那 2 个你接也行不接也行。

### 三个名词：NO / NC / COM

```
COM (Common 公共端)  ──┬── 按下时连 NO
                      │
                      └── 没按时连 NC
NO  (Normally Open  常开)  → 默认断开，按下闭合
NC  (Normally Closed 常闭) → 默认闭合，按下断开
```

**人话翻译：**
- COM = "中间那根"，永远参与
- NO（常开）= 平时不通，按下才通 → 发声盒用这个
- NC（常闭）= 平时通，按下断开 → 急停按钮用这个（断电更安全）

### 怎么找哪两根是 COM + NO？

如果按钮没标，用万用表（设到「通断」档/蜂鸣档）：

```
万用表两根表笔分别接 A / B 针脚
- 没按时蜂鸣 → A-B 是 NC（默认通）
- 按下时蜂鸣 → A-B 是 NO（按下通）
```

或者**没万用表的土法**：

1. 任选 2 个针脚接 LED + 限流电阻 + Pi 3.3V/GND（构成一个测试电路）
2. 按下按钮 —— LED 亮 = 这俩是 COM + NO
3. LED 不亮 = 换另外两个针脚

## Try · 动手验证你真懂了

**用万用表找 COM/NO/NC**（套件里通常有）：

1. 万用表设到「通断/蜂鸣」档
2. 大蘑菇按钮翻底面，任选 2 个针脚接表笔
3. 慢慢按按钮，**听响声**：
   - **按下响、松开不响** → 这俩是 COM + NO ✓ 发声盒用
   - **按下不响、松开响** → 这俩是 COM + NC
   - **永远不响** → 这俩跟开关无关，可能是 LED 那对

记下 4 个针脚的位置，**贴个小标签**（C/N/L/L 之类）—— 几周后你不一定记得清。
